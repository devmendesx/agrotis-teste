### Requisitos para rodar o projeto.

1. Docker
2. Docker Compose
3. Java 11

### Comandos

### Banco de dados

1. Entrar na pasta inicial do projeto e rodar o seguinte comando:
> docker-compose up -d

2. Após instalar o banco de dados na nuvem, poderá rodar o projeto sem problemas.
